# Build Instructions for Semantica Task Engine

## Dependencies

To build and run this project, you'll need:

1. **Rust Toolchain**: Latest stable version (install via [rustup](https://rustup.rs/))
   - Includes Cargo (Rust's package manager)
   - Minimum required: Rust 1.70.0 or newer

## Build Steps

### Development Build

```bash
# Clone the repository
git clone https://github.com/your-username/semantica-task-engine.git
cd semantica-task-engine

# Build the project in debug mode
cargo build

# Run the project
cargo run
